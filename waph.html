Hands-on Labs

Lab 0: Development Environment Setup

I configured my development environment by creating a private GitHub repository and setting up secure SSH authentication for version control. Instead of using a traditional virtual machine, I used WSL2 with Ubuntu 22.04 for lightweight Linux compatibility on Windows. I installed required packages like Apache2, Git, and Sublime, verified the Apache server through localhost access, and committed a templated README with a bio and headshot. This setup provided the foundation for all future labs and projects in the course.

Status: Completed

Lab 1: Foundations of the Web

I explored the structure of HTTP through packet inspection with Wireshark, using curl and telnet to manually send HTTP requests and analyze browser versus raw request headers. I developed server-side applications using both C (CGI) and PHP, including a basic hello-world and an echo script to reflect user input. I compared GET vs POST request behaviors in both structure and security, identifying potential vulnerabilities like XSS and noting proper sanitization techniques such as htmlspecialchars().

Status: Completed

Lab 2: Front-End Web Development

I implemented a complete front-end webpage integrating HTML, JavaScript, CSS, Ajax, jQuery, and Web API calls. Key features included a digital and analog clock using real-time DOM manipulation, email toggling logic, and keypress logging. I demonstrated secure Ajax communications with PHP using both raw JavaScript and jQuery methods, and incorporated external APIs such as JokeAPI and Agify to display dynamic, real-world data. Styling was applied using inline, internal, and external CSS (including W3.CSS) for responsive UI behavior.

Status: Completed

Hackathon 1: XSS Vulnerability Exploration

I tested and exploited multiple levels of Cross-Site Scripting (XSS) vulnerabilities, starting from unsanitized inputs and progressing to regex-based and blacklist-filtered PHP backends. I crafted JavaScript payloads using various injection strategies including <script> tags, DOM-breaking payloads, and onerror image hooks. For each level, I inferred backend logic and demonstrated bypasses, culminating in a secure endpoint using htmlentities() for proper output encoding. This hackathon reinforced both offensive and defensive XSS techniques.

Status: Completed
